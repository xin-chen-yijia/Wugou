# SiQi devlopment tools for Unity3D
封装Unity3D中常用一些功能，避免重复开发，提升开发效率。

Six dev tools  
   |  
   |  ____Examples 一个简单的多人游戏实现;  
   |  
   |  ____MapEditor 地图编辑功能，即在场景中添加模型、特效等内容；  
   |  
   |  ____Multiplayer 基于Mirror的多人游戏代码框架；  
   |  
   |  ____Script 脚本;  
   |  
   |  ____UI 多人游戏中使用的一些UI，可参考借鉴;
